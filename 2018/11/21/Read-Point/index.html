左耳听风专栏笔记

发表于 2018-11-21

读左耳朵耗子博客摘要

技术变现

『我完全没有必要通过打工听人安排而活着，而是反过来通过在公司工作提高自己的技能，让自己可以更为独立和自由地生活。』

『因而，在工作当中，对于那些没什么技术含量的工作，我基本上就像是在学生时代那样交作业就好了。我想尽一切方法提高交作业的效率，比如，提高代码的重用度，能自动化的就自动化，和需求人员谈需求，简化掉需求，这样我就可以少干一些活了。这样一来，我就可以有更多的时间，去研究公司里外那些更为核心更为有技术含量的技术了。』

『要去经历大多数人经历不到的，要把学习时间花在那些比较难的地方』

『25~35 岁是每个人最宝贵的时光，应该用在刀刃上』

『技术和技能的领先』 + 『对技术本质合趋势的敏感度』

『掌握大多数人不能掌握的技能或技术』

『要看一个新的技术是否顺应技术发展趋势，需要将一些老技术的本质吃透，因此在学习过程中要多问自己两个问题，这个技术解决什么？为什么别的同类技术做不到？为什么是这样解决的？有没有更好的方式？』

『找到能体现价值的地方，在一家高速发展的公司中，技术人员的价值可以达到最大化』

『先去大公司学习大公司的技术和成功经验方法，然后再找到高速成长的公司』

『动手能力很重要』

『找到有价值的信息源，到西方世界去』

『利用好时间，投到一些有意义的地方』

技术领导力

吃透基础技术，基础技术是各种上层技术共同的基础

『目的是更好地理解程序的运行原理，并基于这些基础技术进化出更优化的产品。』

『计算机技术太多了，但仔细分析会发现，只是表现形式多，而基础技术不多』

『基础技术学起来可能会感到枯燥无味，但是我还是鼓励你能够克服人性的弱点，努力啃完。』

『基础技术具体分为编程与系统』

编程：C语言、编程范式（面向对象、泛型编程、函数式编程）、算法和数据结构
系统：计算机系统原理（《深入理解计算机系统》）、操作系统原理和基础（《UNIX环境高级编程》《UNIX网络编程》《Windows核心编程》）、网络基础（《TCP/IP详解》）、数据库原理、分布式技术架构

上面这些基础知识通常是不能速成的，需要用一生时间来学习，因为基础上的技术和只是，会随着阅历和经验的增加而有不同的感悟。
